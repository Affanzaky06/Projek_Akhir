# 📈 DSS Investasi Saham Blue Chip  
### Sistem Pendukung Keputusan Menggunakan Fuzzy Tsukamoto & Mamdani

Aplikasi berbasis web untuk membantu pengguna menentukan pilihan investasi saham blue chip terbaik berdasarkan analisis beberapa kriteria seperti return, volatilitas, pertumbuhan laba, utang, dan dividen menggunakan metode Fuzzy Logic.

---

## ✨ Features

- 📊 Analisis saham blue chip secara otomatis
- 🧠 Implementasi Fuzzy Tsukamoto & Mamdani
- 📈 Visualisasi ranking saham dengan Matplotlib
- ⚡ Interactive Web App menggunakan Streamlit
- 🎛️ Pengaturan bobot kriteria secara realtime
- 🗂️ Struktur project sederhana & mudah dipelajari
- 📋 Menampilkan rekomendasi saham terbaik
- 🔍 Perhitungan transparan dan mudah dipahami

---

## 🧠 Metode yang Digunakan

Project ini menggunakan metode:

- Fuzzy Tsukamoto
- Fuzzy Mamdani (Rencana menggunakan ini)
- Defuzzifikasi
- Membership Function
- Decision Support System (DSS)

---

## 📌 Kriteria Penilaian

| Kriteria | Deskripsi |
|---|---|
| 📈 Return | Potensi keuntungan saham |
| 📉 Volatilitas | Tingkat risiko pergerakan harga |
| 💹 Pertumbuhan Laba | Konsistensi peningkatan laba perusahaan |
| 🏦 Utang | Tingkat kesehatan finansial perusahaan |
| 💰 Dividen | Pembagian keuntungan kepada investor |
